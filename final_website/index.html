About the project

About the project

Our project is to understand the collaborative learning methods that are being practiced currently in the smart classrooms and discussions and build a cost effective system that can be used with minimal resources. The goal of our project is to build a system that improves the quality and effectiveness of collaborative learning in smart classrooms or in group discussions and to bring them closer to as many people as possible by making it affordable.

Requirements

Several user studies and interviews were conducted to gather requirements. The following are the main requirements gathered:

  • All users should be able to view the shared content without any obstructions
  • Users should be able to view a 3D model of the shared object
  • The system should enable the driver to manipulate 2D and 3D models with gesture interactions to avoid difficulties faced with traditional keyboard and mouse
  • The system should allow the passengers to track which part the driver is talking about or looking at (ex: pointer or marker)
  • The system should accommodate 2 to 10 users with one user as the driver
  • The system should enable the driver to pull up more information about an object
